# ZeroLink

Example usage of how to run ZeroLink text to SQL models on various inference
frameworks.

* MLX ( Apple Silicon )
* VLLM ( Nvidia GPUs )
* llama.cpp ( CPU )

## Using an Nvidia GPU

This library is tested against Python 3.8+, PyTorch 1.11.
